Constrained Clustering Via Post-Processing

This work proposed the novel approach by converting the output of many clustering algorithm to a cluster fractional allocation matrix (CFAM). Then, takes it as an optimization problem with requirement to satisfy (combined) different types of constraints. In addition, we also shows practical application namely the attribute-level constraint and the (combined) fairness constraint. Summary:

  • Applicable to existing (constrained) clustering solutions
  • Using ILP/Gurobi - powerful optimization model/tool
  • Guarantee constrained satisfaction

Question-Answering using auto labeler and active learning

QA is an important task in NLP. However, the cost of labeling is high as the annotator need to read the whole context and find the appropriate answer. In this repo, we proposes to the following scheme:

  1. Init a pretrain QA model
  2. Inference the dataset using the current model
  3. Export into excel files
  4. Active model select files/questions need to check/answer by human
  5. Human check machine-generated answers and rewrite if necessary
  6. Update model with new labeled data
  7. Test model's performance. Return to step 2 if not satisfy.
